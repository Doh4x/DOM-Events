## Основы DOM-событий

Событие это сигнал что в браузере что-то произошло. Юзер кликнул кнопку, навёл мышь, нажал клавишу, страница догрузилась, форма отправилась. На любое такое событие можно повесить функцию которая отработает в момент когда событие случится. Эту функцию называют обработчиком или листенером.

Событие всегда происходит на каком-то конкретном элементе DOM. Кликнули по кнопке - событие click произошло на этой кнопке. После этого оно ещё всплывает вверх по дереву до document и window, но об этом отдельно.

Способов повесить обработчик несколько. Через атрибут в HTML, через свойство, через addEventListener. Первый вариант давно никто не использует, второй позволяет повесить только один обработчик, третий это рабочая лошадка.

```html
<button onclick="alert('hi')">через атрибут</button>
```

```js
const btn = document.querySelector('button');
btn.onclick = () => alert('через свойство');
```

```js
btn.addEventListener('click', () => alert('через addEventListener'));
```

В обработчик первым аргументом приходит объект события. Из него можно достать всё что нужно - какая клавиша нажата, по какому элементу кликнули, координаты мыши и тд.

```js
btn.addEventListener('click', (event) => {
  console.log(event.type);
  console.log(event.target);
});
```
